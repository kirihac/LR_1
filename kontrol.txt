1. Файли можуть переюувати: збережений у коміті (commited), змінений (modified) та індексований (staged)
2. git add – добавляє файли до гілок
3. git status - перевірка статуса файлів
4. git commit – додати коміт 
5. git init – створення нового репозиторію
6. В коментарях комітів треба писати що змінено в файлі
7. git checkout – перехід до іншої гілки
8. git reset – відміна змін репозиторія
9. git log – перегляд комітів у зворотньому до хронологічному порядку напрямку
10. git status – перевірка статуса файлів
11. За допомогою команди git reflog
12. git checkout commit
13. git branch назва
14. git checkout назва
15. git checkout main
16. git merge приєднувана гілка
